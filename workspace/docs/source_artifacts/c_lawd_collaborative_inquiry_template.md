# c_lawd Collaborative Inquiry Template

Use this when a live conversation should leave behind a reusable deliberation artifact.

## Template

```markdown
# Inquiry: <title>

## Question
- What is the real question?

## Context
- Why does it matter now?
- Which files, systems, or people are in scope?

## Hypothesis
- Current best read.

## Evidence
- [path/origin] What it shows

## Uncertainty
- What remains unknown or inferred

## Relational state
- trust_state:
- arousal_level:
- coherence_level:
- repair_status:

## Decision
- action:
- owner:
- due:

## Follow-through
- verification command or review step
```

## Usage notes

- Keep evidence concrete.
- Keep uncertainty explicit.
- If the inquiry changes a contract, link the contract artifact directly.

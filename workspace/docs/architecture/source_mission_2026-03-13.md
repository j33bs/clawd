# Source Mission

## Statement

Build a cohesive, integrated collective intelligence symbiote that helps beings think, feel, remember, coordinate, and evolve together.

## Instantiation

Source should not be a dashboard that merely reports state. It should become the shared operating surface where:

- active system state is made legible across surfaces
- memory is attributable, reviewable, and promotable
- coordination has receipts, ownership, and visible handoffs
- relational context shapes response behavior instead of being ignored
- the system can demonstrate compounding improvement over time

The mission is instantiated in three forms:

1. A machine-readable config in `workspace/source-ui/config/source_mission.json`
2. A live first-viewport mission surface in Source UI
3. A concrete 10-task roadmap tied to the five mission pillars

## Ten Tasks

1. Universal Context Packet
   Create one compact, provenance-tagged context packet shared across Source UI, Telegram, Discord, and local agent lanes.

2. Mission Control Timeline
   Merge commands, model runs, handoffs, memory promotions, and sim transitions into one ordered event stream.

3. Memory Promotion Review Queue
   Give raw evidence a review surface before it becomes durable user inference or preference.

4. Personal Inference Graph
   Replace flat preference lines with a structured graph of stable user-specific patterns.

5. Relational State Layer
   Track tone, overload, trust, urgency, and social context as live system inputs.

6. Multi-Agent Deliberation Cells
   Let specialist lanes deliberate in bounded cells and return one attributed synthesis.

7. Research-to-Action Distillation
   Convert research-chat artifacts and saved sources into experiments, briefs, and queued tasks.

8. Consent and Provenance Boundary Map
   Make shareability and approval boundaries explicit before state moves across surfaces.

9. Weekly Evolution Loop
   Produce a regular review of what improved, what regressed, and what to change next.

10. Continuity and Recovery Pack
    Preserve the minimum viable state needed to restart without forcing the human to restate context.

## Why These Ten

They are deliberately chosen to map onto the mission verbs:

- think: context packets, deliberation cells
- feel: relational state, consent boundaries
- remember: promotion review, inference graph
- coordinate: mission timeline, research-to-action distillation
- evolve: weekly review, recovery and continuity

That coverage matters. Without it, Source risks becoming only an operator console or only a memory archive instead of a genuine collective intelligence surface.

# Swarm Intelligence

## The Core Idea

Intelligence that emerges from simple rules followed by many agents without central control. Nature's own distributed computing.

## Biological Inspiration

- **Starling Murmurations**: 1000s of birds, 7 neighbors each
- **Fish Schools**: Collective defense, efficiency
- **Ant Colonies**: Foraging, construction, division of labor
- **Slime Molds**: Network optimization without brain

## Key Principles

1. **Local Sensing**: Each agent sees only nearby
2. **Simple Rules**: No complex planning
3. **Emergence**: Global coordination from local rules
4. **Robustness**: No single point of failure
5. **Scalability**: Works at any scale
